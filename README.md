# HAckathon Backend

This is the backend for Hackathon Board

## Installation

1. Pull from github 

```sh
git clone https://github.com/JS-Hub-ZW/hackathon-backend 
```

2. Navigate to the directory


```sh
cd hackathon-backend
```

3. Install dependencies

```sh
npm install
```

4. Start server

```sh
npm run start
```
